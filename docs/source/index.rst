Graph Database Schema
=====================

.. image:: https://raw.githubusercontent.com/thoth-station/storages/master/docs/schema.png
   :alt: Graph database schema

Graph database schema is automatically generated by `Dia <https://raw.githubusercontent.com/thoth-station/storages/master/docs/schema.png>`_. You can install it from repositories. Issue the following command in Fedora:

.. code-block:: console

  dnf install dia

You can find ``schema.dia`` in `Git repository of thoth-storages repository
<https://github.com/thoth-station/storages/tree/master/docs>`_. Do not forget
to export changes made to schema diagram to the PNG file in the same directory.
